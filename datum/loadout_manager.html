loadout_manager
/datum/loadout_manager

Datum holder for the loadout manager UI.

Vars

custom_loadoutA ref to the dummy outfit we're using
dummy_dirThe dir the dummy is facing.
dummy_keyThe key of the dummy we use to generate sprites
loadout_on_openThe loadout list we had when we opened the UI.
menuOur currently open greyscaling menu.
ownerThe client of the person using the UI
tutorial_statusWhether we see tutorial text in the UI
view_job_clothesWhether we see our favorite job's clothes on the dummy

Procs

cleanup_greyscale_menuA proc to make sure our menu gets null'd properly when it's deleted. If we delete the greyscale menu from the greyscale datum, we don't null it correctly here, and it harddels.
clear_slot_greyscaleClears [category_slot]'s greyscale colors.
deselect_itemDeselect [path] item from [category_slot] slot. If it's not a greyscale item, clear the corresponding greyscale slot too.
find_path_in_listFinds [path] in the loadout list and returns the key it's associated to.
get_tutorial_textReturns a formatted string for use in the UI.
init_dummyInitialize our dummy and dummy_key.
loadout_to_outfitTurns our client's assoc list of loadout items into actual items on our dummy outfit. Also loads job clothes into our custom list to show what gets overriden.
rotate_model_dirRotate the dummy [DIR] direction, or reset it to SOUTH dir if we're showing all dirs at once.
select_itemSelect [path] item to [category_slot] slot. If it's not a greyscale item, clear the corresponding greyscale slot too.
select_item_colorSelect [path] item to [category_slot] slot, and open up the greyscale UI to customize [path] in [category] slot.
set_slot_greyscaleSets [category_slot]'s greyscale colors to the colors in the currently opened [open_menu].
toggle_model_dirsToggle between showing all the dirs and just the front dir of the dummy.
