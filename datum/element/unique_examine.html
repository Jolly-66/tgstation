unique_examine
/datum/element/unique_examine

The multiplier on the shuttle's timer Security level (for timer multiplier) Unique examine element. Refactored from the skyrat module by Gandalf2k15. Attach to an atom with some parameters to give it unique text when examined.

Vars

special_descThe special description that is triggered when special_desc_requirements are met. Make sure you set the correct EXAMINE_CHECK!
special_desc_affiliationThe special affiliation type, basically overrides the "Syndicate Affiliation" for SYNDICATE check types. It will show whatever organisation you put here instead of "Syndicate Affiliation"
special_desc_listA list of everything we may want to check based on an examine check. This can be a list of ROLES, JOBS, FACTIONS, SKILL CHIPS, or TRAITS.
special_desc_requirementThe requirement setting for special descriptions. See examine_defines.dm for more info.
toyIf this is a toy. Toys display a message if you don't succeed the check.

Procs

DetachInit our vars If we were passed a examine check that checks the special list, make sure the special list is filled too
